## jira-issue-printer ##

**Requisition**

    `sudo apt-get update`
	`sudo apt-get install nodejs`
	`sudo apt-get install npm`

**Run the program**

	`cd ~/jira-issue-printer`
    `npm install`
    `node server.js`


----------
**Note**

> Before run application, replace jira-issues.xml which is exported from JIRA.
